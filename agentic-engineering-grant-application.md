## Step 2: Details

**Project Details**
> Valdyum is agentic commerce infrastructure on Solana for building, deploying, running, and monetizing autonomous agents. It is positioned less as another AI assistant and more as an execution and commerce layer for Web3 agents: a builder, marketplace, wallet-gated payment flow, on-chain proof layer, and realtime evidence surface in one product.

> The platform combines agent creation, marketplace/fork flows, 0x402-style payment negotiation, wallet-gated execution, on-chain registry/deployment proof, optimized local GPU execution, and dashboard-level transaction evidence. This gives builders a way to turn autonomous-agent execution into verifiable economic activity while keeping performance reliable through local GPU support.

> The current Valdyum monorepo includes a Next.js client, Node/TypeScript server, Rust agent SDK crates, shared packages, Solana dependencies, an Anchor contract workspace, and documented payment/execution flows. The project already includes proof patterns for wallet signatures, payment verification, agent registry logic, marketplace forking, invoices, realtime dashboards, and GPU-assisted autonomous-agent execution.
